# 五亭桥 OBJ 贴图验证页面

## 验证结论

- 修正版 OBJ 包含 188,434 个顶点、280,696 个 UV、186,741 个法线和 377,268 个三角面。
- 所有三角面的顶点、UV、法线索引均完整。
- MTL 正确引用同目录下的 2048 × 2048 PNG 颜色贴图。
- 原 OBJ 额外混有 7 条线段记录，Three.js OBJLoader 会因此把整个对象误判成线段模型，导致贴图无法应用。
- 页面使用的修正版只删除了这 7 条线段记录，没有改动三角面、UV、法线、MTL 或 PNG。

## 打开方式

不要直接双击 `index.html`。请在本文件夹启动一个本地静态服务器，再通过浏览器访问页面。例如：

```sh
python3 -m http.server 8765
```

然后打开 `http://127.0.0.1:8765/`。

`models` 目录中带 `-original.obj` 后缀的是原始文件；不带该后缀的是可正常贴图的修正版。
