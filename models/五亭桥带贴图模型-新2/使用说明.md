# 五亭桥新2 OBJ 修复与贴图验证

- 原 OBJ：356,568 个顶点、752,755 个 UV、371,507 个法线、731,490 个三角面。
- 所有面的顶点、UV、法线索引均完整。
- 原文件额外混有 55 条线段记录，会令 Three.js OBJLoader 将整个对象误判为线段模型。
- 修正版只删除了这 55 条线段，没有改动三角面、UV、法线、MTL 或 PNG。
- 原始文件已保存在源模型目录的 `五亭桥模型-original.obj`。

打开本页面时，请在当前文件夹启动静态服务器：

```sh
python3 -m http.server 8766
```

然后访问 `http://127.0.0.1:8766/`。
